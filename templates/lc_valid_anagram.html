LeetCode Problem:

Valid Anagram

--------

To view my solution on this page, go to the top left corner of the replit (below) and click the file icon and select the correct file.

--------

The strategy for this problem was to use a hashmap data structure to save the number of times each letter occurs in each word. Each word gets its own hashmap. The logic is fairly straightforward. First, if the length of each word doesn't match, they can't be anagrams. If their lengths are equal, iterate through each word, counting and saving the number of times each letter occurs. Finally, iterate through the hashmaps, checking that for each key (each letter), the corresponding number (occurrence count) matches. If they don't match, return false. Otherwise, the function returns true.
