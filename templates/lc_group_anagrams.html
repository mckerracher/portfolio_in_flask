LeetCode Problem:

Group Anagrams

--------

The strategy for this problem is pretty interesting. The key is to use an array of 26 zeroes. Each position in the array corresponds to a letter in the alphabet. This array is then used to count the number of times each letter occurs in a word. The array is then used as a key in a hashmap (the hashmap uses the letter count array as a key and has a list of words as the value) and the corresponding word is added to the list of words matching the letter counts.
