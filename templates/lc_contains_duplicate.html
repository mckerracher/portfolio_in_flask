LeetCode Problem:

Contains Duplicate

--------

The strategy for this problem was to use a hashmap data structure to save each number seen with the index it was seen at. With the hashmap, the solution is straightforward: iterate through nums, and for each number check if it's in the hashmap. If the number isn't in the hashmap, then it's added. If the number is in the hashmap, a duplicate has been found, in which case the function returns true. If the function finishes running without returning true, a duplicate was not found, and so it returns false.
